## Development

0. Install required packages
    ``` bash
    $ sudo easy_install pip
    $ sudo pip install virtualenv
    ```

0. Setup and run
    ``` bash
    $ source env/bin/activate
    $ python run.py
    ```

0. Starts at [localhost:5000](http://localhost:5000)

0. Stop
    - Ctrl+C
    ``` bash
    $ deactivate
    ```
